VWS Auth Tools
==============

Authentication and authorization tools for interacting with the Vuforia Web Services (VWS) API.

Installation
------------

.. code:: sh

   pip install vws-auth-tools

Usage
-----

.. code:: python

   from vws_auth_tools import ...

   pass
